# [소프트웨어] AutoIntern (웹소켓 기반 실시간 데스크톱 AI 인턴) 실행 방안

## 📌 1. 제안 배경 및 필요성
- "책상 차지한 AI, 조교도 인턴도 짐쌌다"는 기사는 이미 현실이 되었습니다.
- OpenAI의 Workspace Agent, WebSockets 기반 Responses API 도입, 그리고 Google의 Gemini Deep Research Agent API(심층 리서치 에이전트) 공개는 단순한 챗봇이 아닌 복잡한 과제를 해결하는 에이전트 시대가 개막했음을 보여줍니다.
- 데스크톱에 상주하면서 사용자의 맥락(웹 브라우저, 로컬 문서 등)을 파악하고, 여러 단계를 거쳐 스스로 연구하고 결과를 정리해주는 "진정한 의미의 인턴" 소프트웨어가 오피스 워커의 핵심 무기가 될 것입니다.

## ⚙️ 2. 핵심 기능 정의
1. **Context-Aware Sidebar (맥락 인지형 사이드바):** 
   - 데스크톱 화면 한 쪽에 작게 떠있으며, 사용 중인 웹페이지나 문서 내용을 인지하여 즉각적인 도움을 제공. (예: 현재 열려있는 엑셀 데이터를 바탕으로 차트를 그릴 코드를 짜주거나 요약).
2. **Deep Research Agent Mode (심층 연구 모드):** 
   - Gemini Deep Research API를 호출하여 "최근 한국 경제 1.7% 성장률의 원인과 베트남과의 경제 협력(원전, 물류) 전망" 같은 복잡한 주제를 스스로 웹 검색, 기사 수집, 분석하여 20페이지 분량의 보고서 초안(마크다운/워드)으로 완성.
3. **WebSockets-based Realtime Interaction (실시간 워크플로우):** 
   - 기다림 없이 스트리밍으로 조사 진행 과정을 시각화(예: 터미널 형태의 진행 바)하여 인턴이 열심히 일하고 있다는 시각적 피드백 제공.
4. **Local File Management (로컬 파일 정리 및 관리):** 
   - 바탕화면에 난잡하게 흩어진 파일들을 분석하여 자동으로 폴더를 생성하고 분류하는 기능(파이썬 스크립트 기반 로컬 툴 호출).

## 🚀 3. 단계별 실행 계획

### Phase 1: 아키텍처 및 최소 기능 제품(MVP) 구현 (1~2개월)
- **프레임워크 선택:** Tauri(Rust 기반) 또는 Electron을 사용하여 크로스플랫폼 데스크톱 앱 개발.
- **핵심 API 연동:** OpenAI WebSocket API(실시간 대화)와 Gemini API(Deep Research)를 백엔드 없이 클라이언트에서 직접 호출(사용자가 자신의 API 키를 입력하는 BYOK(Bring Your Own Key) 방식부터 시작).
- **로컬 맥락 수집:** 클립보드 기반의 텍스트 복사, 스크린샷 캡처(Vision API 연동)를 통해 현재 작업 화면을 에이전트가 인식.

### Phase 2: 에이전틱(Agentic) 워크플로우 강화 (2~3개월)
- **로컬 파일 접근 권한:** 사용자의 명시적인 승인 하에 특정 폴더(예: 다운로드)의 문서를 읽고 정리하는 스크립트 실행 기능 탑재.
- **UI/UX 개선:** 개발자 친화적인 터미널 UI(5x5 픽셀 폰트 등 레트로 감성 적용) 또는 모던하고 미니멀한 채팅 인터페이스 제공.

### Phase 3: B2C 정식 출시 및 마케팅 (4개월 이후)
- **App Store / Windows Store 등록:** 손쉬운 다운로드 및 설치 지원.
- **구독 모델 전환:** BYOK 방식을 유지하되, 자체 백엔드를 거치는 'Pro' 버전 출시. (예: 여러 에이전트 동시 구동, 사전 학습된 전문 페르소나, 클라우드 동기화 기능 포함).
- **바이럴 마케팅:** X(트위터), 틱톡 등에서 "10분 걸리는 리포트 작성을 30초 만에 끝내는 인턴"이라는 쇼츠/릴스 제작 배포.

## 💰 4. 수익 모델 (비즈니스 모델)
- **Freemium + BYOK (Bring Your Own Key):** 소프트웨어 자체는 무료로 배포하고, 사용자가 본인의 OpenAI/Gemini API 키를 입력하여 사용. 이를 통해 서버 비용 제로(0)화 및 폭발적 유저 확보.
- **AutoIntern Pro 구독 (월 $9.99):** 기업용 SSO 연동, 노션/구글 드라이브 네이티브 통합, 클라우드 저장소 동기화, 사용자 정의 단축키(Global Hotkeys) 등 생산성 향상을 위한 고급 기능 제공.
