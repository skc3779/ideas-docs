# [제안 2 상세 실행 방안] B2B 지정학적 리스크 및 유가 변동 예측 AI 솔루션

## 1. 서비스 개요 및 타겟 고객
- **서비스명 (가칭)**: GeoRisk-AI Dashboard
- **타겟 고객**: 유가 변동에 민감한 항공사, 해운사, 물류/수출입 기업의 리스크 관리 부서 및 구매 부서. 원자재 트레이딩을 하는 금융 기관.

## 2. 핵심 기능
1. **실시간 글로벌 뉴스 모니터링 및 감성 분석**: 전 세계 주요 언론, 정부 공식 발표, SNS(X 등)의 텍스트 데이터를 실시간으로 크롤링하여 NLP(자연어 처리) 기반으로 긍정/부정/위기 스코어를 산출.
2. **공급망 타격 예측 맵**: 이란 선박 나포, 호르무즈 해협 봉쇄 등 특정 이벤트 발생 시, 해당 지역을 통과하는 물류망의 지연 확률과 추가 비용을 시각화된 지도(Map)로 제공.
3. **유가 시나리오 시뮬레이터**: "만약 중동 위기가 3개월 지속될 경우" 등 사용자가 특정 파라미터를 입력하면, 과거 유사 사례의 데이터(머신러닝 모델)를 바탕으로 유가 변동 시나리오 및 예측치를 리포트 형태로 자동 생성.

## 3. 수익 모델 (BM)
- **B2B SaaS 구독**: 기업 규모 및 데이터 열람 권한에 따라 Tier별(Basic, Pro, Enterprise) 월/연간 구독료 부과.
- **맞춤형 컨설팅 및 API 제공**: 자사 ERP 시스템에 직접 연동하고자 하는 대기업을 위한 전용 API 토큰 판매 및 구축 컨설팅 비용.

## 4. 기술 스택 및 개발 단계
- **데이터 수집**: Python (Scrapy, BeautifulSoup), 글로벌 뉴스 API (Reuters, Bloomberg 등).
- **AI/ML**: Hugging Face Transformers (다국어 NLP 모델), 시계열 예측 모델(LSTM, Prophet)을 통한 유가 트렌드 분석.
- **프론트엔드/대시보드**: React, D3.js (데이터 시각화), Mapbox (지리 정보 시각화).
- **1단계 (MVP)**: 주요 키워드(중동, 유가, 기뢰 등) 기반 실시간 뉴스 스코어링 대시보드 구축.
- **2단계**: 시계열 예측 모델 통합 및 유가 변동 시뮬레이터 오픈.

## 5. 리스크 및 대응 방안
- **예측 정확도 한계**: 지정학적 이벤트는 블랙스완(예측 불가능한 돌발 변수)의 성격이 강해 AI 예측이 빗나갈 수 있음. 따라서 절대적인 '예측치'보다는 여러 '시나리오(Best, Base, Worst)'를 제시하여 기업이 선제적으로 대응할 수 있도록 보조하는 도구임을 명확히 포지셔닝함.
- **데이터 확보 비용**: 신뢰성 있는 글로벌 경제 매체의 API 사용료가 비쌀 수 있음. 초기에는 무료 오픈 데이터 및 정부 발표 자료를 최대한 활용하여 비용을 절감.
