# [소프트웨어] Q-Visualizer (학습용 양자 컴퓨팅 시각화 및 로컬 시뮬레이터) 실행 방안

## 📌 1. 제안 배경 및 필요성
- "학부생이 100큐비트 양자 시뮬레이션을 구현했다"는 성과(연세대 첫 국제논문)는 양자 컴퓨팅 알고리즘이 일부 초우수 연구실의 전유물을 넘어 학부 교육 과정 및 대학원 기초 연구로 빠르게 확산되고 있음을 뜻합니다.
- 그러나 일반 대학생, 고등학생, 개발자들이 양자 알고리즘(Shor, Grover 알고리즘 등)을 시각적으로 직관적으로 이해하기에는 IBM Qiskit 같은 텍스트 기반 프레임워크가 진입 장벽이 높습니다.
- 이에 레트로 감성(GeekNews의 5x5 픽셀 폰트 등)을 가미하고 모던한 UI(블록코딩 형태)를 통해 양자 회로(Quantum Circuit)를 '블록'처럼 조립하고 시뮬레이션 결과를 화려한 3D 구(Bloch Sphere) 그래픽으로 렌더링해 주는 로컬 소프트웨어 수요가 확실합니다.

## ⚙️ 2. 핵심 기능 정의
1. **Drag-and-Drop Circuit Builder (드래그 앤 드롭 회로 조립기):** 
   - 사용자가 화면에서 Hadamard (H), CNOT, Pauli-X 등 양자 게이트(블록)를 드래그하여 와이어(큐비트 라인)에 배치하면 코드가 자동 생성(Python/Qiskit)되는 노코드 인터페이스.
2. **Real-time 3D Bloch Sphere Rendering (실시간 3D 시각화):** 
   - 특정 게이트를 지날 때마다 각 큐비트의 상태 벡터(Statevector) 확률이 3차원 구면(블로흐 구면) 위에서 어떻게 회전하고 변화하는지 애니메이션(WebGL, Three.js)으로 표출.
3. **Local GPU Acceleration (로컬 GPU 가속 시뮬레이터):** 
   - 20~30큐비트 정도의 교육용 회로는 클라우드 자원 없이도 게이머들이 보유한 개인용 PC의 NVIDIA/AMD GPU(CUDA/OpenCL)로 행렬 연산을 고속 병렬 처리하도록 최적화.
4. **Export to Cloud QPU (클라우드 실제 양자 컴퓨터 전송):** 
   - 사용자가 만든 회로를 버튼 한 번으로 실제 양자 컴퓨터(IBM Quantum Experience, AWS Braket 등)로 전송하여 실제 노이즈가 포함된 결과를 받아볼 수 있는 연동 브릿지 기능.

## 🚀 3. 단계별 실행 계획

### Phase 1: 교육용 데모(MVP) 엔진 개발 (1~3개월)
- **코어 시뮬레이터 엔진 구현:** Rust 혹은 C++를 이용해 상태 벡터(Statevector) 계산을 수행하는 경량 로컬 엔진 개발(또는 기존 Qiskit Aer 등의 오픈소스 시뮬레이터를 래핑).
- **시각화 프론트엔드:** Tauri (웹 기술+Rust 백엔드 결합 데스크톱 앱) 프레임워크를 기반으로 React, Three.js를 사용해 3D 렌더링 UI 제작. (메인 테마로 사이버펑크나 레트로 픽셀 아트 적용).

### Phase 2: 블록 코딩 및 튜토리얼 챕터 추가 (3~5개월)
- **노코드 회로 조립 UI:** 구글 Blockly 등을 변형하여 브라우저 내장형 회로 에디터 연동.
- **튜토리얼 모드 탑재:** "동전 뒤집기부터 시작하는 5분 양자역학" 등 기본 게이트의 개념을 소프트웨어 내부에서 텍스트와 애니메이션으로 한 단계씩 따라 할 수 있는 인터랙티브 학습 챕터 구축.

### Phase 3: B2B/B2G 에듀테크 시장 진입 (5개월 이후)
- **대학 및 특성화고 B2B 라이선스 판매:** 정보 교과, 물리 교과 실습 소프트웨어로 전국 과학고, 영재학교 및 이공계 대학(연세대 등 주요 대학) 학과에 실습용 라이선스 제휴 영업 개시.
- **글로벌 스토어 배포:** Steam 교육 카테고리나 Windows Store에 영문 버전 출시하여 해외 이공계생 공략.
