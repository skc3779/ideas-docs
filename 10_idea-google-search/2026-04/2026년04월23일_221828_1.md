# [솔루션 서비스] SafeAgent Proxy (AI 에이전트 보안 및 개인정보 필터링 API) 실행 방안

## 📌 1. 제안 배경 및 필요성
- 2026년 4월 기준, OpenAI의 Workspace Agent와 Gemini Deep Research API 등 기업 내 자율형 AI 에이전트 도입이 본격화되었습니다.
- 반면, 거대 AI '미토스(Mythos)' 해킹 사고와 국내 기업 '듀오'의 43만 명(연봉, 체중, 가족사항 등 민감정보) 개인정보 유출 사태는 기업의 AI 도입에 치명적인 걸림돌이 되고 있습니다.
- LLM이 사내 데이터베이스에 접근하여 질의응답을 수행하거나 코드를 실행할 때, 사용자가 의도치 않게 민감 정보를 노출하거나 악의적 프롬프트 인젝션으로 데이터베이스를 공격하는 것을 원천 차단하는 중간 보안 계층(Proxy)이 필수적입니다.

## ⚙️ 2. 핵심 기능 정의
1. **Dynamic PII Redaction (동적 개인정보 마스킹):** 
   - 사용자의 프롬프트나 DB에서 LLM으로 전달되는 컨텍스트 내의 주민번호, 연봉, 이름, 주소 등 PII(Personally Identifiable Information)를 실시간으로 `[REDACTED]` 처리.
2. **Intent & Anomaly Detection (의도 및 이상 행동 탐지):** 
   - LLM 에이전트가 호출하려는 API나 실행하려는 코드(예: 과도한 파일 삭제, 전체 DB 덤프 등)의 위험도를 평가하여 차단하거나 관리자 승인을 요구.
3. **HTTP Credential Proxying (자격 증명 프록시):** 
   - GeekNews에서 언급된 'AI 에이전트를 위한 HTTP 자격 증명 프록시'와 유사하게, 에이전트가 직접 API 키를 소유하지 않고 SafeAgent Proxy를 통해서만 사내 인프라에 접근하도록 통제.
4. **Audit Logging & Compliance (감사 로그 및 컴플라이언스):** 
   - 에이전트가 수행한 모든 작업 이력과 마스킹 처리된 로그를 저장하여 GDPR, AI 안전법(최근 입법 지원 논의 중) 등 규제 준수 리포트 자동 생성.

## 🚀 3. 단계별 실행 계획

### Phase 1: 프로토타입 개발 (1~2개월)
- **핵심 기술 검증:** Python(FastAPI) 기반의 API 게이트웨이 구축. 
- **마스킹 엔진 탑재:** 정규 표현식과 경량 NER(Named Entity Recognition) 모델(예: KoELECTRA 기반)을 결합하여 한국어 개인정보(주민번호, 전화번호, 계좌 등) 추출 및 마스킹.
- **연동 테스트:** OpenAI API, Gemini API와 기업 데이터베이스 사이의 릴레이 서버로 동작 확인.

### Phase 2: 웹소켓 및 실시간 워크플로우 최적화 (2~4개월)
- **성능 개선:** OpenAI의 WebSockets 기반 Responses API 등 실시간 에이전트 워크플로우에 병목이 없도록 비동기 스트리밍(Streaming) 파싱 기술 적용.
- **대시보드 구축:** 기업 보안 관리자가 에이전트의 차단 내역, 개인정보 필터링 현황을 한눈에 볼 수 있는 웹 대시보드(React/Next.js) 개발.

### Phase 3: B2B 상용화 및 세일즈 (4개월 이후)
- **PoC(개념 증명) 진행:** 보안에 민감한 금융권, 공공기관, 대기업 사내 벤처 등을 대상으로 무료 PoC 제공.
- **클라우드 마켓플레이스 등록:** AWS, Azure 마켓플레이스에 솔루션을 등록하여 기업들이 손쉽게 자사 클라우드 인프라(VPC) 내에 배포할 수 있도록 지원(On-Premise 및 Private Cloud 호환).

## 💰 4. 수익 모델 (비즈니스 모델)
- **SaaS 구독 (API 호출량 기반):** 월 100만 건 이하 기본료 산정, 초과 시 건당 과금.
- **엔터프라이즈 라이선스:** 온프레미스(On-Premise) 설치형 제공 및 커스텀 마스킹 규칙(회사 자체 문서 번호 규칙 등) 개발 시 추가 비용 청구.
- **보안 감사 리포팅 추가 옵션:** 법적 컴플라이언스 제출용 프리미엄 월간 리포트 판매.
