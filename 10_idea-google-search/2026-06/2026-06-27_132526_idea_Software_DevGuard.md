## 3. 해결 문제와 사용자 페르소나

### 해결하고자 하는 문제 (Pain Point)
1. **신뢰할 수 없는 외부 저장소 클론 위협**: 해커들이 가짜 구인 광고, 가짜 기술 면접(VC 면접), 오픈소스 기여 요청 등을 통해 개발자에게 교묘하게 악성 코드가 삽입된 깃허브 저장소를 클론 하도록 종용합니다.
2. **패키지 매니저의 은밀한 자동 실행 스크립트 악용**: `npm install`, `cargo build` 등 개발자들에게 매우 익숙한 단순 준비 명령을 내릴 때, 패키지 파일 내부의 `postinstall` 훅이나 `patch-package` 설정에 원격 셸(Reverse Shell)이나 기밀 가로채기 스크립트를 삽입해 둡니다. 개발자는 이를 전혀 알지 못하고 실행했다가 모든 계정 정보와 사내 비밀키를 유출당하게 됩니다.
3. **로컬 개발 기기의 물리적 오염**: 개발 장비가 해킹되면 회사의 메인 인프라(AWS 등)와 연결된 키가 유출되어 전사적인 보안 붕괴로 이어지지만, 현재 백신이나 EDR 장비들은 코드 레벨의 훅 동작을 선제 차단하는 데 어려움이 있습니다.

### 사용자 페르소나
- **한민우 (28세, 프리랜서 프론트엔드 개발자)**: Upwork나 다양한 매체를 통해 글로벌 외주 일감을 구하고 있습니다. 일감을 맡기 전 "이 프로젝트 구조를 분석하고 기능을 더해 달라"며 건네받은 수많은 깃허브 레포지토리를 다루어야 합니다. 이 중 해커의 미끼 레포지토리가 섞여 있을까 상시 긴장 상태에 있습니다.
- **최보안 (34세, 스타트업 CTO)**: 사내 개발자들이 오픈소스 기여나 개인 리서치를 위해 수많은 외부 소스코드를 다운받아 로컬 기기에서 빌드하는 것에 대해 불안감이 큽니다. 개발 생산성을 해치지 않으면서도 외부 레포지토리의 위험성을 원클릭으로 검증할 보안 미디엄이 절실합니다.

## 4. 핵심 기능 및 차별점

### 핵심 기능
1. **Isolated Cloud Sandboxed Cloner (원클릭 격리 클론 및 스캐너)**
   - 임의의 깃허브/깃랩 저장소 URL을 입력하면, 사용자의 로컬 컴퓨터 대신 고도로 격리된 DevGuard 클라우드 가상환경(Sandbox)에서 대신 코드를 다운로드하고 패키지를 설치해 봅니다.
   - 빌드 과정에서 일어나는 비정상적인 디렉토리 접근, 알 수 없는 외부 도메인과의 소켓 연결, 자식 프로세스 포크(Fork) 행위를 전수 실시간 모니터링합니다.
2. **Dependency & Post-Install Script Auditor (디펜던시 정밀 탐지기)**
   - `package.json`, `Cargo.toml`, `go.mod` 파일 등에서 비정상적인 버전(가령 유명 라이브러리와 철자 하나만 틀린 Typosquatting), 악용 가능성이 높은 실행 훅(`patch-package`, `preinstall`, `node-gyp` 등)을 식별해 점수화합니다.
3. **Safe Repo-Transfer (안전 검증 코드 이관)**
   - 스캔 및 빌드 테스트 결과 '안전' 등급을 획득하면 악성 스크립트와 은밀한 백도어가 정제된 최종 청정 코드를 로컬 기기에 압축파일이나 자체 안전 지부 저장소 형태로 로드해 줍니다.

### 차별점
- **터미널 세션 시각화**: 클라우드 가상 환경에서 프로젝트가 실제로 설치되고 빌드될 때 어떤 시스템 콜이 일어나고 해커의 터미널과 붙는지(Reverse Connection) 직관적인 상호작용 그래프로 보여줍니다.
- **개발자 채용 시장 네이티브 대응**: 면접용 코드 테스트 제공 기능이 별도로 탑재되어, 구직 개발자가 면접관으로부터 코드를 안전하게 받아 실행할 수 있는 "면접용 보안 코드 검증 세션"을 원클릭으로 열어 줍니다.

## 5. 실행 계획 (MVP -> 확장)

### MVP 단계
- **무료 CLI 및 웹 스캐너**: 저장소 주소를 넣으면 디펜던시 취약점과 `package.json`의 잠재적 위협 스크립트(postinstall 등)를 우선 정적 분석하여 신뢰 점수를 알려주는 초경량 웹 서비스 런칭.

### 확장 단계
- **동적 가상 에뮬레이터 통합**: 도커 컨테이너를 바탕으로 실제 `npm install`이나 `cargo build` 명령어 실행 시의 커널 시스템 콜 흐름을 모니터링하는 정밀 동적 분석(Sandbox Dynamic Analysis) 모듈 탑재 및 개발자용 로컬 안전 CLI 도구 배포.
