# [아이디어 상세 실행 문서] DataVizStudio (데이터비즈 스튜디오)

## 1. 아이디어명
- **DataVizStudio (데이터비즈 스튜디오)**: 복잡한 HTML 테이블 데이터 자동 추출 스토리텔링 숏폼 비디오 빌더

---

## 2. 제안 범위 및 카테고리
- **대범위**: 유튜브 콘텐츠 및 미디어 (YouTube & Media)
- **중분류/소분류**: AI 자동화 채널 및 데이터 시각화 미디어 콘텐츠 시스템

---

## 3. 해결 문제와 사용자 페르소나

### 해결하고자 하는 문제 (Pain Point)
1. 위키피디아, 주식 시장 보고서, 기후 공시 등 복잡하고 방대한 테이블 형태의 수치 데이터들이 인터넷에 무궁무진하지만, 일반 사용자는 긴 표를 직접 들여다보는 것을 극도로 싫어하고 흥미를 잃음.
2. 데이터 중심의 차트 레이싱(Chart Racing)이나 유익한 시각화 뉴미디어가 인기가 매우 높지만, 수천 개의 셀 데이터를 손수 수집 정제하고 모션 그래픽 툴로 렌더링하기에는 극도로 고된 편집 노가다와 리소스 손실 수반.

### 사용자 페르소나
- **페르소나 1**: 황민혁 (28세, 데이터 저널리즘 인포그래픽 숏폼 크리에이터): 전 세계 주식, 부동산, 역사 순위 통계 표를 긁어 화려하고 동적인 3D 차트 숏츠 비디오를 매일 5개 이상 제작해 업로드하고 조회수 광고 수익을 내고 싶어합니다.
- **페르소나 2**: 유은경 (34세, 공공기관 홍보 교육 담당 주임): 기관이 배포하는 지루한 한글/엑셀 보고서 통계 표들을 스마트폰 사용자용 초감각적인 비디오 인포그래픽 카드로 3분 만에 자동 전환하여 홍보하고 싶어합니다.

---

## 4. 핵심 기능 및 차별점

### 핵심 기능
1. **원스톱 HTML 테이블 투 비디오 파이프라인**: 사용자가 웹페이지 링크를 입력하거나 통계표 스크린샷을 업로드하면 AI가 Simon의 변환 알고리즘을 활용해 데이터를 즉각 JSON으로 무결점 정제합니다.
2. **모션 그래픽 숏폼 비디오 자동 렌더링**: 정제된 데이터를 기반으로 연도별 변화 추이를 동적 3D 모션 그래프 및 자이언트 레이싱 차트로 자동 비주얼 렌더링하고, 통계가 지닌 극적인 인사이트(예: '역사상 가장 빨리 성장한 도시 TOP 10')를 요약한 AI 보이스 나레이션 음성과 자막을 탑재한 1분 숏츠 영상을 2분 만에 자동 빌드합니다.

### 차별점
- 단순 템플릿 활용이나 수작업을 넘어서, **인공지능의 깊은 다단계 추론(Reasoning) 및 데이터 무결성 검증 자동화가 실시간 결합**되어 사용자 경험을 획기적으로 향상시킵니다.

---

## 5. 실행 계획 (MVP -> 확장)

### MVP 단계
- 위키피디아 통계 링크를 받아, 데이터가 실시간 움직이는 간단한 레이싱 바 차트(Racing Bar Chart) 비디오 클립을 파이썬 내부 렌더러로 동적 자동 추출하는 가이딩 웹 툴 완성.

### 확장 단계
- 국내외 데이터 리서치 포털, 언론 미디어의 경제 기사 파트너로 연동하여 통계 보도 자동 생성 플러그인 독점 패키지로 납품 확대.
