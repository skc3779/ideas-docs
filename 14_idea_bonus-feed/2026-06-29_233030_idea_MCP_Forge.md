# 아이디어 상세: MCP-Forge

## 1. 아이디어명
MCP-Forge (기업용 AI 컨텍스트 빌더)

## 2. 제안 범위 및 카테고리
- **분야**: 소프트웨어 (Software)
- **카테고리**: 개발 도구 (DevTools)

## 3. 해결 문제와 사용자 페르소나
- **해결 문제**: 기업들이 자사의 내부 데이터(DB, API, 사내 문서)를 AI 에이전트(Claude, GPT 등)에 연동하려면 복잡한 파이프라인(RAG 등)을 구축해야 함. 최근 등장한 MCP(Model Context Protocol) 표준이 이를 혁신적으로 쉽게 만들 수 있으나, 비개발자나 IT 부서가 빠르게 배포하고 관리할 수 있는 도구가 부재함.
- **타겟 사용자**: 기업 내 IT 인프라 관리자, 사내 AI 도입을 주도하는 PM 및 엔지니어 팀장.

## 4. 핵심 기능/차별점
- **노코드 MCP 서버 구축**: 드래그 앤 드롭 방식으로 사내 데이터베이스(PostgreSQL, MySQL), 사내 협업툴(Notion, Jira)을 연결하여 자동으로 MCP 서버를 생성.
- **접근 권한 관리 (RBAC)**: AI 에이전트가 접근할 수 있는 데이터의 권한을 부서/사용자별로 세밀하게 제어.
- **실시간 모니터링 대시보드**: AI 모델이 MCP를 통해 어떤 데이터를 얼마나 자주 호출하는지 가시성 제공.

## 5. 실행 계획 (MVP -> 확장)
- **MVP 단계**: 
  - 가장 수요가 높은 PostgreSQL/MySQL 및 Notion 기반의 MCP 서버 자동 생성 기능 제공.
  - 생성된 MCP 서버를 Claude Desktop 등 클라이언트와 원클릭 연결하는 설정 파일 제공.
- **확장 단계**:
  - 기업용 SSO(SAML) 연동 및 완벽한 보안 터널링 기능 제공.
  - 다양한 SaaS(Salesforce, Zendesk 등)용 프리빌트 MCP 커넥터 마켓플레이스 오픈.

## 6. 수익 모델
- **B2B SaaS 구독**: 
  - Basic: 소규모 팀을 위한 클라우드 호스팅 MCP 서버 (월 $49)
  - Enterprise: 온프레미스 배포 옵션 및 세밀한 권한 제어, SLA 보장 (Custom Pricing)

## 7. 일정/리소스 계획
- **1~2개월 (프로토타이핑)**: MCP 프로토콜 핵심 라이브러리(Node.js/Python) 기반 노코드 래퍼 개발.
- **3~4개월 (베타 테스트)**: 3~5개 B2B 스타트업 대상 무료 도입 후 피드백 수집 및 보안 취약점 점검.
- **5~6개월 (정식 런칭)**: B2B 유료 플랜 오픈 및 세일즈 파이프라인 구축.

## 8. 실현 가능성 점수 및 근거
- **점수**: 5점
- **근거 요약**: "MCP는 AI를 위한 RSS"라는 평가처럼 기술적 복잡도는 낮으나 파급력은 매우 큼. RSS 생성 도구들이 초기 웹 시대에 큰 성공을 거두었듯, MCP 서버 생성 플랫폼 또한 현 시점에서 개발 리소스 대비 성공 확률과 시장 수요가 압도적으로 높음.
