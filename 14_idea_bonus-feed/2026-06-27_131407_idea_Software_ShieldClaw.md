# [아이디어 상세 실행 문서] ShieldClaw (실드클로)

## 1. 아이디어명
- **ShieldClaw (실드클로)**: LLM 에이전트 자율 방어용 시뮬레이션 레드팀 및 프롬프트 방화벽(Prompt Firewall) SaaS

---

## 2. 제안 범위 및 카테고리
- **대범위**: 소프트웨어 (Software)
- **중분류/소분류**: 보안 (Cybersecurity) 및 개발 도구 (DevTools)

---

## 3. 해결 문제와 사용자 페르소나

### 해결하고자 하는 문제 (Pain Point)
1. **에이전트 인프라의 위협 노출**: LLM 에이전트가 데이터베이스, 이메일, 메신저 등 외부 채널에 직접 연동될 때, 임의의 악의적 프롬프트 주입(Prompt Injection)을 통해 기업 기밀이 노출되거나 시스템 권한이 도용될 리스크가 상존합니다.
2. **사전 검증의 부재**: 개발자는 본인이 구축한 프롬프트 시스템이 외부 공격에 얼마나 강한지 정량적으로 평가하기 어렵습니다. 수작업 테스트는 비효율적이고 완성도가 떨어집니다.
3. **실시간 트래픽 제어 불가**: 한 번 배포된 LLM 프롬프트는 런타임 중에 비정상적인 우회 질문이나 탈옥(Jailbreak)을 실시간으로 식별하고 걸러내는 전용 보안 미들웨어가 필요하지만, 현재는 대안이 드뭅니다.

### 사용자 페르소나
- **김시큐 (34세, DevSecOps 엔지니어)**: 금융 및 엔터프라이즈 레벨에서 자사 서비스에 LLM 챗봇과 업무 에이전트를 통합하는 프로젝트의 보안 검증을 책임지고 있습니다. 프롬프트 해킹으로 인한 개인정보 유출을 방어해야 하는 과제를 안고 있습니다.
- **박빌더 (29세, 풀스택 AI 엔지니어)**: 오픈소스 에이전트 프레임워크를 사용해 프로덕트를 만들었으나, 이메일 연동 에이전트가 악성 이메일을 수신해 자동 실행될 때 발생할 프롬프트 인젝션 취약점을 자동으로 점검받고 싶어 합니다.

## 4. 핵심 기능 및 차별점

### 핵심 기능
1. **Automated Red-Teaming Simulator (자동화 해킹 침투 테스트)**
   - 배포 전 LLM 에이전트의 API 엔드포인트 및 프롬프트 인스트럭션을 제공하면, 수천 개의 해킹 에이전트 봇(Hack-Bots)이 침투 테스트를 가합니다.
   - 프롬프트 인젝션, 개인정보 강제 인출, 백도어 실행, 탈옥(Jailbreak) 등 다양한 가상 시나리오를 주입해 점수를 산정합니다.
2. **ShieldClaw Prompt Firewall SDK (실시간 프롬프트 방화벽)**
   - 에이전트가 응답하기 전 실시간으로 입출력 텍스트를 고속 필터링하는 파이썬/Node.js SDK를 제공합니다.
   - 인젝션 성향이 강하거나, 기밀 데이터 패턴(API Key, 이메일 주소, 사내 정보 등)이 검출되면 에이전트 호출을 즉각 차단하고 경고를 보냅니다.
3. **Vulnerability Scorecard & Report (보안 취약점 성적표)**
   - 해킹 테스트 완료 후 취약한 인스트럭션 라인을 하이라이트하고 안전한 대안 프롬프트를 자동 제안합니다.

### 차별점
- **에이전트 특화 방어 아키텍처**: 단순 텍스트 검사를 넘어 에이전트의 '도구 호출(Tool Calling)' 및 외부 API 결합 흐름에 최적화된 독자 필터링 모델을 내장하고 있습니다.
- **검증된 데이터셋**: 실제 오픈소스 해킹 챌린지(`hackmyclaw.com`)에서 수집된 실제 공격 시나리오와 수만 개의 공격 프롬프트 말뭉치 데이터를 학습한 로컬 판별 모델을 활용하여 성능 저하(오탐)를 줄입니다.

## 5. 실행 계획 (MVP -> 확장)

### MVP 단계
- **주요 타겟**: AI 빌더들이 가볍게 자사 에이전트의 프롬프트와 API를 올려 테스트할 수 있는 무료 웹 기반 취약점 스캐너 개발.
- **핵심 기능**: 기본 탈옥 프롬프트 데이터셋 1,000종을 활용한 자동 스캔 및 프롬프트 대체 수정본 제안 기능.

### 확장 단계
- **SDK & SaaS 연동**: 백엔드 API 서버 앞에 즉시 미들웨어 형태로 끼워 넣을 수 있는 `shieldclaw-py` 패키지 배포.
- **엔터프라이즈 솔루션**: 대형 조직에서 프롬프트의 보안 라이프사이클을 추적하고 SOC(보안관제센터) 대시보드와 통합할 수 있는 연동 서비스 제공.

## 6. 수익 모델
1. **Freemium (개인 빌더/스타트업)**: 월 10회 이내의 에이전트 자동 레드팀 테스트는 무료로 제공하되, 이상 검출 상세 보고서와 패치 제안은 유료로 전환.
2. **SaaS 구독형 (성장형 팀)**: 월 99달러 ~ 299달러. 실시간 프롬프트 방화벽 API 호출 수에 비례한 과금 및 정기 레드팀 위협 모니터링 자동화.
3. **Enterprise 라이선스**: 온프레미스(On-premise) 설치 및 LLM 내부 도메인 전용 미들웨어 구축 패키지 판매.

- **실현 가능성 점수**: **4.8 / 5.0** (매우 높음)
- **근거**: 
  - 최근 Fernando Irarrázaval이 진행한 `hackmyclaw.com` 챌린지처럼, 이메일 수신 기반 AI 에이전트에 대해 2,000명의 해커가 가한 6,000번의 해킹 시도는 보안 실무자들의 뜨거운 관심을 유도했습니다.
