## 제안 범위

### 1. 소프트웨어 (Software)
사용자의 문제를 해결하거나 업무 효율을 높이는 '도구' 중심의 영역입니다.

| 카테고리 | 설명 | 주요 키워드 |
| :--- | :--- | :--- |
| **B2B SaaS (기업용)** | 기업의 운영 효율을 높이는 구독형 소프트웨어입니다. | 협업 툴, ERP, 프로젝트 관리, CRM |
| **개발 도구 (DevTools)** | 개발자의 생산성을 높이거나 인프라 관리를 돕습니다. | CI/CD, 코드 리뷰, API 관리, 모니터링 |
| **보안 (Cybersecurity)** | 데이터 보호 및 네트워크 위협에 대응하는 기술입니다. | 제로 트러스트, 개인정보 암호화, 엔드포인트 보안 |

### 2. 데이터 및 인공지능 (Data & AI)

| 카테고리 | 설명 | 주요 키워드 |
| :--- | :--- | :--- |
| **생성형 AI 응용 (LLM App)** | 대규모 언어 모델을 특정 도메인에 맞게 활용합니다. | RAG(검색 증강 생성), 에이전트, AI 비서 |
| **데이터 인프라 (DataOps)** | 데이터의 수집, 저장, 가공 프로세스를 자동화합니다. | ETL 파이프라인, 데이터 레이크하우스, 파이프라인 모니터링 |
| **산업 특화 AI (Vertical AI)** | 의료, 금융, 제조 등 특정 산업군에 최적화된 AI 모델입니다. | 진단 AI, 이상 탐지(Anomaly Detection), 알고리즘 트레이딩 |
| **MLOps** | AI 모델의 학습부터 배포, 유지보수까지의 전 과정을 관리합니다. | 모델 모니터링, 버전 관리, 자동 배포 |

| 카테고리 | 설명 | 주요 키워드 |
| :--- | :--- | :--- |
| **마켓플레이스 (Platform)** | 특정 유무형의 상품을 거래하는 중개 플랫폼입니다. | 전문 서비스 매칭, 리커머스, 하이퍼로컬 커머스 |

### RSS FEED 수집 가이드
- 1차로 `RSS FEED` 포멧을 참고하여 수집하며 `trends_data_1차_[일시].json` 파일을 생성한다.
  - 수집항목 : `제목`, `링크` 만 수집한다.
- 2차로 `1차로` 수집된 json 파일을 이용해 직접 사이트를 방문하여 내용을 분석 후  `trends_data_2차_[일시].json` 파일을 생성한다.
  - 수집항목 : `제목`, `링크` 및 `요약정보`를 추가 수집한다.
- 정리된 `trends_data_2차_[일시].json` 파일을 심층 분석하고 아이디어 제안에 활용한다.
